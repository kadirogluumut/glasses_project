# 👓 Arduino Smart Glasses (Distance Alert System)

This project is a DIY Arduino-based smart glasses system that detects distance and provides audio feedback using a buzzer.

## 📌 Project Overview
The smart glasses use a distance sensor to measure how close an object is. When an object comes closer, the system triggers a buzzer that changes its beeping frequency depending on the distance.

This is a simple wearable tech prototype for learning Arduino, sensors, and real-time feedback systems.

## ⚙️ Features
- 📏 Distance measurement using sensor
- 🔊 Buzzer alert system
- 📉 Beeping frequency changes with distance
- 👓 Wearable prototype design
- 🧠 Beginner-friendly Arduino project

## 🧰 Components Used
- Arduino UNO (or similar board)
- Ultrasonic Distance Sensor (HC-SR04)
- Buzzer
- Wires
- Battery / Power supply
- DIY glasses frame or mount

## 💡 How It Works
1. The sensor measures the distance of nearby objects.
2. Arduino processes the distance data.
3. If an object gets closer, the buzzer starts beeping faster.
4. If the object moves away, the beeping slows down or stops.
